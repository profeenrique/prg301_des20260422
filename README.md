# 🟢 DES 20260408 PRG-301 miércoles

## DES01: ¿Sobrevivirás al día escolar?
Crea un programa que determine tu nivel de energía según la cantidad de horas que dormiste.

El programa debe pedir la cantidad de horas que dormiste. Luego, debe indicar tu nivel de energía según la siguiente tabla:

| Horas de sueño | Nivel de energía |
|----------------|------------------|
| menos de 5     | Modo zombie      |
| entre 5 y 7    | Funcionando      |
| más de 7       | Full energía     |

**⬇️ Entrada:**
```
¿Cuántas horas dormiste? 4
```

**⬆️ Salida:**
```
Modo zombie
```

## DES02: Nivel de gamer
Crea un programa que clasifique a un jugador según su puntaje.

El programa debe pedir el puntaje del jugador y luego indicar su nivel según la siguiente tabla:

| Puntaje | Nivel |
|---------|-------|
| menos de 1000 | Jugador novato |
| entre 1000 y 3000  | Jugador pro |
| más de 3000 | Leyenda |

**⬇️ Entrada:**
```
Ingresa tu puntaje: 3500
```

**⬆️ Salida:**
```
Leyenda
```

## DES03: ¿Te alcanza para el bajón?
Crea un programa que indique qué puedes comprar según el dinero que tienes.

El programa debe pedir el dinero disponible y luego indicar qué puedes comprar según la siguiente tabla:

| Dinero disponible | Qué puedes comprar |
|-------------------|-------------------|
| menos de 2000     | Te alcanza para un bebida  |
| entre 2000 y 5000 | Te alcanza para un completo |
| más de 5000       | Te alcanza para un completo, una bebida y un postre |

**⬇️ Entrada:**
```
¿Cuánto dinero tienes? 4500
```

**⬆️ Salida:**
```
Te alcanza para un completo
```

## DES04: Control de notas
Crea un programa que analice una nota final y muestre un mensaje según el resultado.

El programa debe pedir la nota final y luego indicar el mensaje correspondiente según la siguiente tabla:

| Nota final | Mensaje |
|------------|---------|
| menos de 4.0 | Reprobaste |
| entre 4.0 y 5.9 | Aprobaste |
| entre 6.0 y 6.9 | ¡Seca! |
| igual a 7.0 | ¡Perfecto! |

**⬇️ Entrada:**
```
Ingresa tu nota: 7.0
```

**⬆️ Salida:**
```
¡Perfecto!
```
